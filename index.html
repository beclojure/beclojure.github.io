Welcome to Clojure in Belgium ... why not start with a little Clojure right now...


(+ 1 2 3 4)

The above code is the a minimal but complete program in Clojure!

Clojure code is always in the form of a list within parentheses:

( function [one or more arguments] )

In the live example above, we have the function '+' and the arguments 1, 2, 3 and 4. The +, like the other maths functions, acts accumulatively. Maths functions are not operators baked into the language. You can try some others: -, /, *


(<= 1 2 3 4)

Like other languages, many functions in Clojure also have specific arguments 


(rand 1000)

The code evaluator has a magic trick where it can be put on a loop :) 


(map #(rand %) (range 1 4))

map is an example of a function that operates on a sequence. Reading from right to left range produces a list of numbers from 1 to 3. #(rand %) is a shortcut to call rand with the argument (%).
